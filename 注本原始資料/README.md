<h1>注本原始資料</h1>
<p>這裏搜集的，是一些杜詩的重要注本的原始資料。這些原始資料，除了注本原文之外，還附帶校記、後設資料。後設資料的加入，是要爲將來用程式來處理原始資料鋪平道路。</p>
<p>因爲版權問題，當代學者的著作，如蕭滌非主編的《杜甫全集校注》，雖然我打算把它們全部數字化，但在我有生之年，還不能提供數字文本。<p>
<h2>計劃中要納入的注本</h2>
<p>在未來的幾年內，我計劃納入以下注本：</p>
<ul>
<li>蕭滌非《杜甫全集校注》</li>
<li>謝思煒《杜甫集校注》</li>
<li>下定雅弘、松原_朗《杜甫全詩訳注》</li>
</ul>
<p>在這三大套書之後，我想加入一些宋、清的注本；但這可能要等個十年八年了。</p>
