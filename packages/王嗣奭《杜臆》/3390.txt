王嗣奭《杜臆》214,PDF267,243,PDF3.130【評論】公詩喜用「萬事」，原非套語。男兒一身，萬事屬焉，身已黃髮，萬事已矣，止此殘生，隨白鷗以去而已。結語乃失意中自寬之詞，亦知公之流淚非爲一身之私也。◯公之入蜀從關塞來，關塞阻而向瀟湘，非本意也，故用「轉」字。【按語】「萬事」見於：萬事益酸辛 => 0328 奉贈鮮于京兆二十韻萬事終傷不自保 => 0695 蘇端薛復筵簡薛華醉歌萬事隨轉燭 => 1349 佳人成都萬事好 => 1853 五盤萬事反覆何所無 => 1989 杜鵑行大向交遊萬事慵 => 2110 暮登四安寺鐘樓寄裴十歎息人間萬事非 => 2410 送韓十四江東覲省萬事盡付形骸外 => 2685 相從歌萬事干戈裏 => 2920 倦夜蕭條萬事危 => 3036 傷春五首萬事已黃髮 => 3390 去蜀萬事糾紛猶絕粒 => 3521 寄常徵君人生萬事無不有 => 5297 可歎百年從萬事 => 5522 遣悶
