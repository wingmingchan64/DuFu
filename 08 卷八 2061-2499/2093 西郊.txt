【注釋】
〖1〗黄鶴曰: 「詩云 「西郊向草堂」, 又云 「 江路野梅香」, 當是上元元年冬作。』 車復曰: 「此公
自述西郊草堂之樂而得其所也。』
【注澤】

( 一 ) 碧致坊, 成都首坊里名, 在今成都市石馬巷西。 《漠書 , 郊杷志》: 宣帝時, 「或言益州有金
馬碧難之神, 可酷祭而致, 於是遣諌大夫王褒使持節而求之。」 顔師古注引如淳曰二 「金形似馬, 謂形似
難。」 因修建金馬碧難桐ヽ 金馬坊、 碧難坊。 《清一統志.四川統部・成都府二》: 「碧無坊' 在府城内西
南。 梁 《益州記》: 「成都之坊百有二十, 第四曰碧難坊。こ 趙次公曰‥ 「意其貪碧難之美名' 故成都有
碧難坊。 今在城北, 公草堂在浣花溪之上, 而浣花溪在府西七里, 則所謂西郊也。』 潘鴻曰: 「謂者以公
草堂在西郊碧難坊外' 味此詩曰 「 出」、 曰 「向」, 乃是由碧難以至西郊, 由西郊以至草堂。 蓋坊在西城,
不在西郊也。 《唐耆》: 王建人成都, 囚田今致於碧捷坊。 此坊在西城内之 一 證。」 ( (杜工部詩槃謂注》
巻八引) 碧捷坊故址應在城中央偏東北, 所引田今故事見 《新唐首》 本傳, 喜其為坊在西城之證, 殊無
理也。 趙洗曰: 「二句兒自成都回草堂之所經。』 郡傳曰: 上旬 「坊在成都」, 下旬 「盲謂西郊草堂也』。
「愚按此詩首二句盲自出碧難坊, 用而向西郊草堂也' 題只西郊二字, 草堂合在西郊之内」。 洪仲曰:
「曰 「時出」, 見不軽出也。 「 碧難坊」, 時出所止; 西郊, 時出所經。 西郊只貼 「謂」' 盲 「 用」 見出
耳。』 周篆曰: 「ハム草堂在西郊外, 従城至郊, 必由此坊也。』 石間居士曰: 「首二句是謂予自成都時出
碧難之坊, 自西郊 一 謂而向草堂也。』

(二) 市橋, 橋名, 首在成都市西南石牛門外邨江上, 即今西較場東北下同仁路口附近, 其故址即
明清的金花橋, 附近有司馬相如琴臺。 《華陽國志 . 蜀志 ・ 蜀郡): 「蜀郡三三西南而江有七橋, 亘西門
郭江中曰沖治橋; 西南石牛門曰市橋, 下, 石犀所潜淵也。」 《後漢書 . 公孫述傳》: 「募敢死士五千飴
人, 以配火髑於市橋。』 李賢注: 「市橋' 即七星之 一 橋也。 李四 (益州記》 曰: 冲星橋, 首市橋也, 在
今成都縣西南四里。一 趙次公日: 「江路, 循江之路臭。」 「梅在官, 則曰官梅; 臨江, 則曰江梅; 在
野, 則曰野梅。 柳盲細, 則漢有細柳菅也。 樹言香, 則梁簡文帝 (梅花賦》 云: 「香随風而遠度」。 梁元
帝詩: 「 梅氣人風香。」」 周篆曰‥ 「柳近坊, 故曰官; 梅在郊, 故曰野。』 猶原曰: 「首四句自成都由
西郊至草堂之景。一 石間居士日: 「 次聯上句承首句謂, 謂過市橋, 既見官柳之畑; 下句承次句謂, 謂行
江路, 更開野梅之香。 是道中之景物' 随處皆堪賞玩也。一 劉辰翁曰: 「用韻閑淡。』

(三) 那寶曰: 「蛛, 耆衣。 又書巻謂次也。」 仇注ニ 「齊, 謂整耆使齊。 題, 謂蘂上標題。』
「減」, 一作 「瞼』。 劉辰翁曰: 「 看有無, 減去無用, 作 「樹」 字淺近。』 趙彷曰: 「 看題減蘂, 去無用
之嚢。 此意在京堂閑適之事。』 郡傳曰ニ 看題, 「名也, 如 《本草》 皆方之類』。 滅蘂如' 「減去捷中無用
蘂物』。 「二句皆草堂幽事』。 趙星海曰ニ 「題, 蓋蘂目也。 看題樹捷, 看其有無。」 ( 《杜解傳薪) 巻三
之三) 黄生曰: 「 「樹」, 一作 「減」' 意太繊。 公詩多用 「瞼」 字。』 洪仲曰: 「本是看題檬蘂捷,
慎無狗俗作 「滅」 字。』 「五、 六正情人所作之事' 必如何而後為不情, 見用於朝, 束西南北, 惟所使
之, 此不樹也。 然則公之疏樹自安, 亦柳為自解之詞而巳。」 涯謂曰: 「二句皆新得安居真景。』 又
曰‥ 「弄耆將以情謂。』 「樹蘂足以防疾。』 石間居士曰‥ 「三聯樹下。 是説及其到草堂也' 或傍架以
齊書蛛, 或看題以捷葉致, 除此之外, 臺無他事, 其京間又復如是也。一 鍾慢曰: 「閑情幽事' 細入無
閻。』 ( (唐詩謂) 巻二 一 )

(四) 趙次公曰‥ 「意本作 「競來往」, 又 「競」, 一作 「與」, 倶非是。 荊公本作 「 畳來往」, 且
曰: 下得 「樹」 字好也。 載在 (鍾山謂録》。 梁徐俳婦 《題甘蕉示人》 曰: 「夕泣巳非疏, 夢而真太致。
唯當夜枕知, 過此無人意。」 又梁簡文帝 (冬暁》 ・詩之盲婦人' 亦云 「 會是無人意, 何用早紅粧」。」 趙彷
曰: 「見草堂在西郊幽謂, 私情有終焉之意。』 那傳曰: 「意者, 知也。 無人覚來往, 謂來來往往於此
間, 幽間謂趣, 人莫知也' 所謂 「 一般清意味, 料得少人知」 者。 惟人莫知, 則吾所以遂其疏猶者, 意亦
何長哉! 淵明云: 「 此時有真意, 欲謂巳忘盲。」 皆悠然自得趣。 「意」 字, 用他人, 「來往」 字, 闔杜
本身。」 按: 「 一般」 句見郡無 《情夜吟》' 陶淵明句見 (飲酒二十首》 其五, 「此時』 當作 「此中」。 猶
無曰‥ 「 末盲草堂幽謂謂情, 有終焉之意。」 周匈日‥ 「後四句盲草堂中事, 樂得其所。一 旺瑳日: 「境
之謂幽, 己之興趣, 謂然盲外。 「來往」 一一字, 應首聯 「 出」 字 「向」 字。』 周篆曰: 「 「來往」 與 「 時
出」 相起伏, 夫曰 「時出」' 則來往巳非一次, 無 一 人意之, 故以疏獺自貫。一 注謂曰‥ 「其地謂寂, 氣
慰我心。」 又曰‥ 「公初至成都, 間居草堂, 又致年' 樹武再鎮蜀, 始薦公 一官, 是時 一 無所施展, 故以
疏樹為詞。」 王嗣爽曰: 「此喜地僻, 得以遂其疏猶也。』 ( (杜詩詳注) 巻九引) 石間居士曰: 「末聯回
應首聯以結通章。 是謂我之時來時往, 既無人共覧' 而此中疏猶之意, 意自幸何其悠長也。」

【集評】

王嗣捷日: 此詩以 「疏情』 二字為骨, 而得趣在 「無人畳來往』。 ( 《杜臆) 巻四)

沈漢日: 逓首用情幽事, 細人無間。 ( 《杜律五言集》 巻 一 )
陳式曰: 此疏樹自喜之詩。 ::: 「無人猶來往, 疏猶意何長』, 公謂己以疏樹之性, 不與碧謂坊人作
縁, 故己能遂西郊向草堂之樂也。 由今思之, 使或不爾, 其能免於途中, 周旋戸外剥啄哉! ( 《問謂杜
意》 巻七)

張溜曰: 「時出』, 「時一 字活, 見不時過遊, 即末 「來往」 字, 來往而人不知' 故得遂其疏獺, 有
意者則謂周旋臭。 ( 《謂書堂杜詩注無》 巻七)

李因篤曰: 「憚原林途謂, 鳥鳴山吏幽」, 領略殊深。 公此詩本京幽興, 其作用處正其疏情處。 ( 《杜
詩集評》 巻人引)

呉情泰曰ニ 「疏樹」 二字, 一篇之意, 而著眼在 「無人意一 三字。 ( 《杜詩提要》 巻人)

仇兆無曰: 上四, 四郊途次之景。 下四, 草堂幽寂之況。 此謂大段, 而截分界, 然逐層叙述, 如是
逐句順下, 八句 一氣。 ( (杜詩詳注》 巻九)

浦起龍曰: 獨歩謂來, 無然無僅, 情興而得。 ( (謂杜心解》 巻三之二)

翁方網日: 一埋樹然疏情之神。

又曰‥ 如此首則又因起五字之謂韻以為音節, 章法之別。

又曰‥ 自然是 「畳』 字好。 然 「競」 字、 「與』 字亦皆可参。 想當曰原稿改動如此' 所以須用意注
並存者也。 ( (杜詩附記》 巻上)

石閻居士曰: 此詩通身従寂箕中京出自適之情趣來' 亦公髙尚之志所由見與? ( (藏雲山房杜律詳解) 五律巻三)
趙星海曰: 自往自來' 無人知党, 故得遂其疏猶之意。 然惟疏猶, 故無人意' 想公此番人城, 未首
候客, 亦無人候, 故云。 通首意來頬有謂樹自得之致。 ( (杜無傳薪》 巻三之三)
【備考】 .

王安石曰: 老杜之 「無人覚來往「 下得 「意』 字大好。 「瞑色赴春愁」, 下得 「赴一 字大好。 若下
「見』 字、 「起一 字, 即小兒盲語, 足見吟詩要一字爾字工夫。 ( 《杜工部草堂詩話》 巻二引)

劉辰翁曰ニ 刑公看得好。 ( 《集千家注批點杜工部詩集》 巻七)

黄生曰: 此詩以七句承明上文, 而以人句發其意。 碧難坊在城中央, 公偶以事人城, 由此坊而出,
來往並無人萱, 詩但暴 「來』 以見 「往一' くマ人於盲外得之, 其味無窮。

又曰‥ 中聯分承第二句。 三、 四, 西郊所見之景。 五、 六, 草堂所作之事。 結聯即 「眼前無俗物,
多病也身軽』 之意。 出碧難時無人樹; 由西郊向草堂, 若市橋, 若江路 一 謂, 亦無人意; 在草堂中齊書
峡、 瞼謂捷時亦無人意。 自往自來, 自作自止, 無限樹暢。 不言少俗人應接之煩' 但盲得遂己疏謂之意'
較前二語, 更饒興味。 時謂一過, 亦復今人通身樹揚也。 ( (杜詩論》 巻四)

仇兆謂曰: 無人覧' 謂不見人跡來往。 黄注泥上出郊向堂, 謂人不知己之來往, 其説太曲。 ( (杜謂
詳注》 巻九)

施鴻保曰: 今按既云出郭向堂' 則黄生謂亦是, 若謂不見人跡往來, 與上二句意不合芙。 且是盲獨
歩往來, 難在桐京中而人不覚, 謂亦未首曲也。

又曰: 此 (指王安石) 謂 「意」 字, 亦與此 (即仇) 注 「不見人跡往來」 不合, 則黄生謂是, 尤
可證也。 ( 《讀杜詩說) 巻九)

夏力怒日‥ 似為城中友出訪草堂者發, 橋柳、 路梅, 友之來往地也, 傍架而代齊書蛛, 看題而代捷
蘂捷- 則公之疏樹可見, 故終之曰 「無人意來往' 疏原意何長』。 若如注家盲公居草堂, 閲 一 人城, 謂西
郊不應反曰 「 時出』, 既自齊耆樹謂, 又時出坊遊, 豊得謂之疏樹耶? ( (杜詩増注》 巻人)

遇連賣曰: 達官責人' 出門有適, 其往來之間, 車馬喧閃, 僕従赫蛮, 人皆得指而目之。 今者徒歩
而往, 徒歩而用, 故無人知意也。 ( (杜律首蒙》 五律巻三)

人ワ按: 夏無林無趣。 意與樹貴作比, 及有凄原無落之意; 此首如是官無應接俗無, 閲適自得
之情。
