( 一 ) 西山, 在今四川省松潘縣, 為眠山主峰, 海抜五千五百多米, 因山頂無年積雪, 故猶雪山,
雪嶺, 又因在成都西, 故又猶西山、 西嶺。 三奇戊' 唐軍戌名。 在今四川都江堰市。 《新唐書 ・ 地理志 `
六》: 「 (彰州導江縣) 西有璽崖關; 有眠山、 玉堕山。 有鎮翻軍, 閲元中萱。 有白沙守捉城。 有木瓜
戌, 三奇戌。」 《魯唐書 . 韋泉傳》: 「無乃今鎮謂軍使陳泊等' 統兵萬人出三奇路。」 王應麟 《困學紀聞》
引此二條' 又云: 「 《西南備邊録》 所謂三奇省也。」 「當従魯本三奇為是。」 一作 『三城戌』。 (新唐書・
高適傳》: 「姶上皇束還' 分則南為南節度, 百姓翻於謂度, 而西山三城列戊。 適上疏曰‥ 「則南如名束
西川, 其實 一 道。 ……」 帝不納。』 此三城則指眠山地區的松州 (治今松潘縣)` 截州 (治今理縣), 保州
(治今理縣新保關), 為成都府西北屏障, 與吐蕃接界, 歴來是唐, 蕃戦中的沖要之地, 軍事地位十分重
要。 唐代設則南道西山防謂使府, 承則南西山節度魯制, 駐成都。 銭箋ニ 「按西山三城, 界於吐蕃, 為蜀
邊要魯, 厘見杜詩, 正不必作三奇。 此穿璧之過耳。」 萬里橋, 又名篤泉橋, 府河大橋。 在四川成都市南
錦江上, 是古代乗舟車出成都的起程處。 (元和郡縣翻志・則南道.成都府.成都縣》こ 「萬里橋架大江

日: 「西山在松、 維州之外。 維州, 今之威州是也。 冬夏有雪, 號為雪山' 所以控幣吐蕃之處。 詩 (按:
當作 「時」) 吐蕃方人冠, 故須防戊臭' 高適上疏可證三城置戌之始。 魯本作 「 三年」, 非。」 謝無曰:
「雪嶺三城之戊' 時所備也; 清江萬里之橋' 吾所居也」。 張性曰: 「盲野望之際, 遠則見雪南, 近則見
南浦。』 仇注‥ 『臨橋而望三城, 近慮吐蕃。』 夏力怒曰ニ 「起二句従 「野」 宇發端。』 石間居士曰:
『 一 以見翻關之盗賊謂盛, 一 以見異地之飄零極危也。」 日本津阪孝棹曰‥ 「 起句西望, 悲邊害無巳, 思
戊兵之魯; 次句南望思謂, 有 「股勤翻西路, 此去赴長安」 之感。 蓋野望之際, 遠則西山三城列戊, 近

也。」 日本津阪孝猶曰: 「風塵' 盲兵乱。 公有四弟, 故曰諸弟。 此承 「 三城戌」, 曰 「 海内風塵」, 不止
蜀中芙。 下句承 「 萬里橋」, 望郷之情, 泣自吊也。 一聯盲因蓑宇援翻, 與骨肉隔無, 孤身南莫, 各自天
涯, 恨望恨恨, 惟有揮嶼耳。」 夏力怨曰: 「硬借 「 一身」 南宇開出下文。」

(三〉 張性曰: 「遅暮, 晩遅之意; 謂埃, 小流細崖也。』 『如此衰老多病, 官位不顕, 略無報主
之截。」 郡博曰: 下句 「此句甚重, 蓋高適疏翻三城戌, 朝廷不納, 公阻於建明, 故此太息。一 似不用牽
批髙適事。 張滑曰‥ 「君國之傀。』 黄生曰‥ 「 「供」 宇工甚, 遅暮之身, 尚思截力朝廷, 豊意第供多病
之用? 此自悲自恨之詞。」 仇注‥ 『五、 六西自慨。一 日本津阪孝無曰: 「前聯思家, 此則思國, 蓋此生
老衰, 夙志磋蛇, 男兒七尺之嘔, 徒供病度曰, 無復有能為。 昔如通籍朝廷, 恭猶近侍之官' 未魯有尺寸
之功報答聖主之恩, 不勝遺憾之至, 翻漸傀自截也。 夫以魯契輩人, 而使老棄閲猶, 非唯不形怨望, 且倦
捲如此, 忠厚之至也。 「 遅暮」, 「泪埃」, 虚實魯, 亦曰翻重翻。」 李長祥曰‥ 「遇暮多病' 下一 「供」

(四) 張性曰: 「聯欲出散懐, 而眼中人事日見南條, 適以増其傷感也。」 謝木从曰: 『但見世翻人

「 出郊」, 「極目」, 倒結有力。』 陳醇儒曰: 「人事者, 即通指以上所青弟隔、 身遙、 城戊, 老病之事, 此

張艇曰‥ 目無時無' 徒自不能為惰而巳。 然則公之悌涙者, 豊以 一 身翻旅之故哉? 實傷世乱而爛人
南山, 遠播江湖之外, 切懐廊廟之憂' 史謂人憐其忠者, 其以是夫! ( 《杜律本義〉 巻二)

郡傳日‥ 邁乃出郊時謂望, 祇見四野翻條, 日甚 一 日, 百姓之謂發巳弊, 朝廷之惠政未流, 世翻人
窮, 於吾情何以堪之。 臆! 天涯悌涙, 公豊 一 身悌涙平? 通篇意自首句出。~明年吐蕃果陥京師' 西山諸
州皆没! ( 《杜律無解》 七律巻上) .

顧施禎曰: 此詩要看 「時一 宇、 『日」 宇, 望野為題, 不是翻一時所望, 而盲人事, 亦非翻一 日而
喜, 乃常常野望, 日非 一 日。 上六句不就著望講, ロハ得截述養局。 宣到七句' 鮎出正題, 八句拮出本意。
( (杜工部詩疏解》 巻 一 )

陳之翻曰‥ 杜謂不獨後四句, 先用 一 聯虚虚而列, 而後乃牧明之, 即前四句亦然' 如此篇之類是也。
全首經逸疏宕, 與 《進艇》 詩同一風格, 妙在不借一事' 不飾 一 詞, 而自然悲涼。 (《杜工部七盲律詩注》
巻二)

黄生曰‥ 起二句即極目所見, 結處乃鮎明之。 「南浦」 句截近景, 然以 「萬里』 為名, 則目不至而
心巳至之' 此所以與上句相猶。 而人事南條, 蓋不止目前所見而巳。 三, 四骨肉喋離之戚' 五、 六闔廷疏
遠之懐, 此則人事之民切者。 跨馬出郊之際, 極目魯心, 宜首及此。 ( (杜詩諭》 巻九)

呉農祥曰: 梱懐吐謂, 可詠可歌。 ( (杜詩集評》 巻十 一 引)

仇兆繁曰: 此因野望而寄慨也。 上四, 野望感懐, 思家之念。 下四, 野望撫時, 憂國之情。 ( 《杜詩
詳注》 巻十)

査慎行曰‥ 可猶高渾。 前四句第五宇皆敢且相犯, 學者宜忌。 ( (杜詩集評) 巻十 一 引)

浦起龍曰: 國患家離, 雨雨繋心。 「三城戌一' 提憂國, 「萬里橋一, 提思家。 三、 四, 頂次句' 恩家
之切也。 五, 六, 頂首句, 憂國之仇也。 題中 「望』 宇意, 皆暗藏在内, 七, 鮎清, 八, 無牧。

又曰: 中四, 思家憂國' 分中有合。 ( (謂杜心解〉 巻四之 一 )

夏力怒曰: 三、 四野中情思, 五、 六野中人物, 七、 八方寓出望來, 八通結上文也。 ( (杜詩増注》
巻八)
毛張健曰: 起聯有野望中。 二聯述心事, 結聯分猶, 局法最為明晰。 妙在三` 四価根首聯, 而下句
法截翻, 意實側注, 故五、 六即従四句生出, 何其融治也。 ( (杜詩譜樺》 巻 一 )

石間居士曰: 此詩無意横放, 大氣盤旋' 層層頂貫, 語語神接, 一聯深 一 聯, 宜至末聯倒鮎出野望
來, 猶能更深 一 層作牧結。 可謂眼髙於頂, 力大於身, 集中惟此種筆墨尤見少陵本色, 自來謂者皆尋常看
過, 惜哉! ( 《藏雲山房杜律詳解》 七律巻上)

趙星海曰‥ 上二聯作爾層, 家國互南。 五、 六切 一身, 家國渾南。 七貼八牧' 然國歩多翻, 皆由人
事所致, 兵革搜翻, 人事益見南條, 故五` 六自慨, 非徒虚空作感無語也。 ( 《杜解傳薪》 巻四之 一 )
