=蕭====================================================
蕭滌非主編《杜甫全集校注》2.1025,djvu1202

【異文、夾注】
〖宮〗宫
〖栖〗棲

【注釋】
〖1〗黄鶴曰‥ 「當是乾元元年春在諫省時作。』 補注: 「由人明而退朝而謂院' 再従院中回謂宮景,
井院中之事, 然後結到出披, 時自晩芙。』
【注澤】

( 一 ) 謂次公曰ニ 「衛宏 《漢謂儀》, 使夜漏起, 宮衛傳呼以為備。 陸睡以其所載為未詳, 謂傳呼
淺, 則在貢不若夜之遠也。」 「春旗, 盲羽衛也。』 洪仲曰‥ 「 淺之為言鮮也。 即七律 「 査漏稀聞高閣報」
稀字。」

(二) 顧宗曰: 「退朝, 謂院, 漉指諸臣盲。」 「曰散' 曰迷, 其出春天晩景。』 浦注: 「院, 即左
披。」 命謂曰: 「按 「文昌雜録》 云, 杜甫 《晩出左披》 云 「 退朝花底散, 謂院柳退迷」。 乃知唐朝殿亦
種花柳, 今殿廷惟謂植椀轍, 有巌毅之氣。 按此知唐制, 松門蘂樹, 惟以表位, 乃古謂棘之遺意, 而此外
則花柳扶疏, 亦自兒天家富貴也。』 ( 《春在堂全耆・茶香室四抄》 二十八)
・ (三) 顧窟曰‥ 「雪融, 則春己深。』 張滑曰: 「去殿低' 盲雲近傍殿, 見傍殿之髙。」 退連安曰‥
「樓在城上, 故雪融而謂。』 黄周星曰‥ 「 「櫻雪融城」, 「宮雲去殿」 與 「欲鶏棲」 三字, 皆倒錯句法也。
惟老杜長千用此。』 ( 「唐詩快》 巻九)

(四) 《詩・王風・君子子役》: 「鶏棲千謂, 日之夕是。』 「曹耆・羊枯傳》二 「其嘉謀謂議' 皆
焚其草, 故世莫聞。』 顧窟曰: 「此宜盲焚諫草, 避人聞知, 不欲避名耳。』 鍾慢日ニ 「 「 避人焚諫草」,
元老話頭。」 ( 「唐詩謂》 巻十五) 那傳曰: 「騎馬者出左液也。 欲鶏棲, 則其出晩芙。』 翁方綱曰: 「焚
諫草與前章右封事, 相為開合消納。』

【集評】

陳輔之曰: 「明明有封事, 敵間夜如何?』 此幸而得之, 坐以待旦之意。 「避人焚諌草, 騎馬欲鶏
棲。』 即所謂 「嘉謀嘉献, 人告謂后, 子外曰‥ 斯謀斯献, 惟我后之徳也』。 ( 「陳謂之詩話》)

張謂曰: 此詩中皆承春字盲, 末句方及晩出, 甚有次第。 《春宿左省》 一首, 惟首句見春, 下皆承暮
字盲, 可悟雙化之妙。 ( (謂書堂杜詩注解》 巻四)

翁方謂曰‥ 《晩出左披》 ~詩, 較之 (春宿左省》 篇, 尤為含蓄謂謂, 評家或謂其退食之風度' 或
稲其得諫臣之謂, 皆未得其深處。 蓋其日晩出左液, 乃純是 一 片謂主之仇, 融結而出, 所以覧得其伎齊斑
之際査漏殊淺也。 散而迷者, 非因身在柳邊, 正因心在君側耳。 ( 《石洲詩話》 巻 一 )

【備考】

杜田曰: 謂宗至徳初, 子美為拾遣' 巻段為補明。 或問二人段賢? 余曰: 子美賢。 或曰‥ 何以知之? 曰: 以其詩知之。 子美之詩曰: 「避人焚諌草, 騎馬欲鶏棲。」 又曰‥ 「明明有封事, 政間夜如
何?」 巻之詩曰‥ 「聖朝無明事, 自謂諫書稀。」 至徳初, 安史之乱方則, 上皇在蜀' 朝野騒然, 果無明
事時邪? ( 「巻溪漁隠叢話》 前集巻六引 《老杜補遣》)

劉辰翁曰: 焚諌草者, 不欲人知也。 然使人知其焚謂' 是猶欲知也。 如焚稿' 亦避人, 正是鮎破古
事。 無限懇歓, 此事君當然之體。 ( 「集千家注批鮎杜工部詩集》 巻四)

陳式曰‥ 「避人焚諫草, 騎馬欲謂棲。」 蓋設詞以問晩出之故, 而非實有之事也。 使必如須溪云云,
夫既公然見之詩篇, 而焚不且甚於不焚哉? 固哉髙安, 固哉, 不通之謂也。 ( 《問齋杜意》 巻四)

注謂曰: 公在左披, 曰 「 明朝有封事』, 曰 「避人焚諫草」, 則宗朝張后、 輔國授政, 廣平死謂, 安
史除凶未滅, 當見謂封事, 諫草者多芙。 公於三司推問後, 傷弓之鳥, 欲諌不敢, 而公平生忠赤, 欲謂不
能, 公於未離諫垣, 必時時恐君遺失, 随時建白, 以匡不逮, 又不肯顛暴諸景, 以彰君過也。 二語是公居
官時其實經濟, 公自表出 「宿左披》、 《出左披》 詩。 若巻補明詩云: 「聖明無明事, 自寛諫紙稀。」 蛙以
寄公, 非公意突。 ( (樹人堂謂杜詩》 巻六)

朱亦棟曰: 杜子美 (晩出左披》 詩: 「避人焚諫草, 騎馬欲謂棲。』 謂日之將夕, 謂巳欲棲也。 或
引後澳謂曰: 「車如鶏棲馬如狗, 疾悪如風朱伯厚。」 則與晩出意不合臭。 然 「欲謂棲』 三字, 到底費
解。 按潘安仁 (寡婦賦》: 「謂登棲而或其。」 謂疑 「 欲」 字即 「敏」 字之訛耳。 ( 《群書札記》 巻五)

=名====================================================
《名家注評全唐詩》387,PDF445

【異文、夾注】
〖晝刻傳呼淺〗晝刻傳呼淺。[趙汸：傳呼淺，謂傳呼在晝不若夜之遠也。]*
〖避人焚諫草〗避人焚諫草。[劉須溪：焚諫草，不欲人知也。避人而焚，並掩其迹矣。]*

【注釋】
〖春旗〗青旗。
〖焚諫草〗喩爲官謹愼。諫草，向皇帝上諫表的草稿。
〖雞栖〗《詩》：「雞栖於塒，日之夕矣。」用雞栖以點晚出。

【評論】
王嗣奭《杜臆》：杜詩妙在氣象，此於退食時，能寫出委蛇氣象。
仇兆鰲《杜詩詳註》：首二朝班之景，三四退歸院，五六院中之景，七八出院歸舍。晝刻記日，春旗記時，中四皆言春景。唐時殿廷，多植花柳，故初退朝班，從花底而分散，各歸院舍，至柳邊而遮迷。樓在城上，故雪融而濕。殿高逼云，故去殿若低。及焚草而出，時値雞栖，見日已晚矣。
=奭====================================================
王嗣奭《杜臆》63,PDF116,74,PDF1.175

【評論】
老杜詩妙在氣象，此於退食時寫出委蛇氣象。◯「樓雪融城濕」，非觸目寫不出，而亦人所不及著眼，公每用此取勝。
