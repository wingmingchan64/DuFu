2008 戲爲雙松圖歌[韋偃]

天下幾人畫古松，畢宏已老韋偃少。
絕筆長風起纖末，滿堂動色嗟神妙。
兩株慘裂苔蘚皮，屈鐵交錯迴高枝。
白摧朽骨龍虎死，黑入太陰雷雨垂。
松根胡僧憩寂寞，厖眉皓首無住著。
偏袒右肩露雙腳，葉裏松子僧前落。
韋侯韋侯數相見，我有一匹好東絹，
重之不減錦繡段。
已令拂拭光凌亂，請公放筆爲直幹。

【注釋】
〖畢宏已老韋偃少〗畢宏亦畫工也。
〖絕筆長風起纖末〗仲尼作《春秋》，絕筆於獲麟。《長笛賦》：其應清風也，纖末奮䔠。
〖滿堂動色嗟神妙〗趙云：滿堂，如滿堂爲之不樂。《左傳》：使者色動而言肆。
〖偏袒右肩露雙腳〗《金剛經》：偏袒右肩，右膝著地。◯趙云：因畫胡僧而紀詠之，故用佛書字焉。《張良傳》載：四皓之厖眉皓首，衣冠甚偉。《楞嚴經》云：名無住行，名無著行。公摘其字而合用之也。然唐有中興間氣集，載鄭賢詩云：高僧無住著，何日出東林。賢與公同時人，莫知孰先用也。
〖重之不減錦繡段〗《四愁詩》：美人贈我錦繡段。◯趙云：不減者，不虧也。本出《左傳》：不爲末減。其後晉人多言某人不減某人。
〖已令拂拭光凌亂〗謝惠連：清波時凌亂。◯趙云：梁吳均《行路難》曰：未央采女棄鳴箎，爭見拂拭生光儀。謝眺《和劉繪詩》：赬紫共彬駮，雲錦相凌亂。

【注釋】
〖1〗黄鶴曰= 「詩云: 「我有一匹好束絹」, 指束川梓州謂溪謂而盲, 則此詩作於成都, 従蓄沈在
上元元年。」 韋堰, 京兆 (人ウ陜西西安) 人, 寓居於蜀。 朱注: 「(名董記》ニ 韋襄子鵬工山水、 高
僧、 奇士、 老松、 異石, 筆力勁健, 風格高暴。 人知鵬善馬, 不知松石更佳。」
【注欅】

( 一 ) 皐宏, 當時著名壷家。 銭箋‥ 「封演 《閏見記》: 睾宏, 天賓中御史' 善貫古松。 後見張
操, 於是閣箪。 《名畳記》: 大暦二年' 為給事中, 董松石於左省用壁' 好事者皆詩詠之。 改京兆少尹為
左庶子。 樹木改歩愛古, 自宏始也。」 此二句將韋鵬與當時著名蜜家撃宏並謂, 推崇其宣技。 郡長薇曰:
「 起法又別。」 ( 《杜詩鏡鐙》 巻七引) 夏力怨曰‥ 「蒼葬落筆' 目無千古。』

(二) 鮑箪, 謂査成而欄筆。 長風起謂末, 形容此董之箪カ勁健, 風格高皐。 上旬, 形容此董所産
生的藝術魅力。 東漢馬融 (長笛賦》: 「其應清風也' 鐡末奮箱。』 爾句評謂韋堰此董之神妙。 張遠日:
「此段淫以畳松叙起, 不専指。」

(三) 屈鐵, 比喩松枝臀曲而色黒如鐵。

(四) 爾句窩壷中古松之状, 其形態其氣勢其精神, 均遍似真松。 前句承 「雨株惨裂」 句而來, 因
其皮裂, 故松幹之剥蝕如龍虎朽骨, 以 「白』 領起, 突出此董之筆法枯淡; 後句承 「屈鐵交錯」 句而
來, 因其枝髙回旋, 故其氣之陰森如雷雨下垂, 以 「黒」 領起, 並與上 「白」 謂照, 突出此董之笠墨濃
郁。 用句造句奇警, 超邁奇古。 張艇曰: 「白擢' 盲蘆之枯淡處。 黒人, 盲蘆之濃潤處。」 朱注: 「 (史
記索用》: 極南為太陽, 極北為太陰。 皮裂, 故幹之剥蝕如龍虎骨朽; 枝回, 故氣之陰森如電雨下垂。」
張遠日: 「此段正盲蜜松之状, 題之内象也。」 夏力怒曰: 「奇巧典切, 而 一 氣渾成, 故妙。』 楊注‥
「造句奇警。」
(五) 胡僧, 西域僧人。 憩, 休息, 鹿眉皓首, 長眉白頭。 住著, 佛教用語, 即執著。 仇注:
「 (榜巌經》‥ 名無住行, 名無著行。』 趙次公曰: 『因韋胡僧而紀詠之, 故用佛書字焉。』

(六) 偏祖右眉, 佛教徒身披袈裟, 祖露右肩, 以表示恭敬。 以上四句描繪其寅中所繪之人物。 朱
注: 「 《金剛經》: 備祖右眉, 右膝着地。 (長水經疏》: 祖, 肉祖也。 西方俗儀, 見王者必肉祖, 示非
敢有犯, 佛教亦随此用。 然此以表將荷大法之重措耳。』 蒔金式曰‥ 「寓人定僧宛然。』 ( (杜詩鏡詮》 巻
七引) 張遠曰: 「此段紋松下所坐之僧, 題之外象也。」 夏力怒曰: 「 四語空中鮎竄, 最妙。』

(七) 韋侯, 即韋堰。 敢, 快、 速。 束絹, 王士槙日: 「蜀璽亭縣有鶏溪' .縣出絹, 謂之謂渓絹,
亦名東用, 子美詩 「我有 一 匹好束絹」 是也。」 ( 《池北偶談》 巻十人) 銭箋: 「呉曹 《漫録》: 束絹,
關束絹也。 庚肩吾 (答武陵王賛絹啓》 日: 關束之妙' 潜熾阻其巻鱧。 按: (唐志》: 束川陵州, 土貢
謂溪絹。 謂注云二 即此詩 「束絹」。』 錦繍段, 精美的絲織品。 張衡 《四愁詩): 「美人贈我錦舗段。』

(人) 光浚乱, 指素用所展時光影浚乱的様子。 宣幹, 指樹幹挺披的松樹。 張遠曰: 「上詠宣巳
完, 而此拓 一 歩, 正見其査之妙。』 仇注‥ 「此索韋宣松也。』

【集評】

王嗣爽曰: 起來二句極寛謂' 而忽接以 「用筆長風起殲末」, 何等箪力! 至於描寓雙松止四句, 而
冥思玄謂, 幽事深情, 更無剰譜。 後人 「胡僧」, 曹冥如超, 更有神氣。 然韋之畳松, 以屈曲見奇, 亘便
難工。 一匹束用, 長可二丈, 汝能 「放筆為宜幹」 平? 所以戯之也。 ( (杜臆》 巻四)

黄生曰‥ 「 用株」 四句, 謂松之膚幹枝葉, 造語極奇極確。 以松為主, 胡僧特董中鮎綴, 舟鶴之類
耳。 萱格有賓主, 故叙意有軸童。 題中 「戯」 意, 亦見末段。 ( (杜詩説》 巻三)

李因篤曰: 老筆奇氣, 足排萬人。

又曰: 似古樂府。 ( 《杜詩集評》 巻五引)

呉農祥曰: 宜起亘落' 無首無尾。 ( (杜詩集評》 巻五引)

注謂曰‥ 極口眼中見所董之雙松, 如句句藏鋒, 我絹必須如此畳乃妙, 謂之贄壷詩也可, 謂之乞貫
詩也可。 ( (樹人堂謂杜詩》 巻九)

沈徳潜曰‥ 突亢起, 不妨平接, 如 「堂上不合生楓樹』, 下接 「聞君掃郁赤縣圖』 是也。 平調起,
必須用瞥謂接, 如 「天下幾人宣古松』' 下接 「鮑謂長風起鐡末』 ・是也。 學者於此求之, 思過半芙。 ((唐
詩別裁集》 巻六)

浦起龍日‥ 首四句, 總統寶之。 次四句, 細幕其状。 「裂皮」、 「回枝」, 寓出鱧幹。 「白催』, 寓枯梗
掬折處。 「黒人」' 鴬風針奉鬆處。 又次四句・ 鮎綴法。 「無住著」, 神理都現。 「僧前落』' 空寂所然。 末
五句, 於諸題壷詩鰭法又出 一 奇, 與 「 心平愛芙, 遇不謂芙」 同 一 意境, 蓋傾倒之極也。 蒼注認真, 作素
董解, 便療。 ( 《讀杜心解》 巻二之二)

夏力怒曰: 此種詩即昌黎所謂 「光談萬丈長」 者也。 ( 《杜喜注》 巻七)

劉鳳詰曰‥ 少陵不倭佛, 抑又深通佛理。 三… ( 「 松根』 四句) 鮑妙好機鋒, 知自有證人處。 ( (杜工部詩話》 巻 一 )

又曰: (韋厘雙松圃歌》' 開口便説 「天下幾人臺古松「 便兒韋董鮑頂。 鮎明雨株, 即吠其皮裂,
玩其枝回, 「白擢朽骨龍虎死' 黒人太陰雷雨垂』 二謂, 玄構幽思, 真有鬼神之助。 蓋皮裂則幹巳剥如'

【評論】
起來二句極寬靜，而忽接以「絕筆長風起纖末」，何等筆力！至於描寫雙松止四句，而冥思玄構，幽事深情，更無剩語。後入「胡僧」，窅冥靈超，更有神氣。然韋之畫松，以屈曲見奇，直便難工。一匹東絹，長可二丈，汝能「放筆爲直幹」乎？所以戲之也。
PDF2.166
《通》云：「『白摧』一句，言畫之枯淡處。『黑入』句，言畫之濃潤處。此聯超邁奇古。」
