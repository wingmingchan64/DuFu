1755 泥功山

朝行青泥上，暮在青泥中。
泥濘非一時，版築勞人功。
不畏道途永，乃將汩沒同。
白馬爲鐵驪，小兒成老翁。
哀猿透卻墜，死鹿力所窮。
寄語北來人，後來莫匆匆。

【集評】

魯復曰: 言行此山, 若將陥按之可畏, 賦也。 ( (謂杜詩愚得》 巻六)

王嗣謂日: 古云‥ 成州有人景樓, 泥功山與風凰臺居其二。 公詩止盲其魯, 不盲其勝, 何也?

又曰: 山上有泥功廟, 石魯古怪, 豊以謂勝耶? ( (杜臆》 巻三)
唐元按曰: 泥功名既奇, ハム之此解, 當由目撃。 夫山路崎嘔, 用之可耳。 以泥故版築, 既所難喩,
縦有之, 不過 一時。 今以此名山. 則相沿不知幾何或月, 至今未己也。 且此必低隋之地, 何以名山? 詩
中有 「哀猿透如墜』 之句, 又似非低處所宜, 安得身至其地 )考之。 ( (杜詩按》 巻 ) )

陳式曰: 山下泥魯, 其來久遠, 版築魯人功, 版築徒魯無用, 泥然其常耳。 至因所遭泥魯, 生出將
來旧没相伺之按, 此語亦只帯下, 故下第意泥魯苦吠, 垂戒後人, 經此當慎, 猶云不因泥魯而所行人之
意。 ( (間齋杜意》 巻六)

張謂曰: 此紀行泥得之苦而井戒後人也, 版築, 以布道路。 成老翁, 謂力疲也。 透欲墜, 謂猿即上
亦下墜, 鹿以隋泥而死。 按非得意之詩, 然必如此, 方形出泥山之瞼。 ( (謂書聖杜詩注魯》 巻六)

按金式曰: 記地之作, 撲老如古樂府。 ( (杜詩鏡魯》 巻七引)

浦起龍曰二 前四, 宜起。 「版築) 句, 不脱 「功」 字。 時或有命工填築之事, 因紀其魯也。 後八,
都従泥謂上生按。 「不畏」、 「乃將』, 借景以洩其慣。 而畑玩作意, 如因將次息足, 特地志慨, 以束全局。
然聯, 勧世謂, 恰是到頭語也。 ( 《謂杜心解》 巻 一 之三)

【評論】
《一統志》：「山在成縣西二十里。」古云：成州有八景樓，泥功山與鳳鳳臺居其二。公詩止言其濘，不言其勝，何也？又云：「山上有泥功廟，石像古怪。」豈以稱勝耶？
